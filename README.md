# 🐕 DOG FOOD CUTTOR - 사료 커터 모터 제어기

## 1. 개요

| 항목 | 내용 |
|------|------|
| MCU | ABOV A31G11x (ARM Cortex-M0+) |
| 클럭 | HSI 40MHz |
| 개발환경 | IAR Embedded Workbench 7.5 |
| 용도 | 반려동물 사료 커터 모터 제어 |

## 2. 핀 할당

| 핀 | 기능 | 방향 | 설명 |
|----|------|------|------|
| PE0 | PWM30AA | 출력 | 모터 PWM 제어 (20kHz) |
| PE1 | KEY | 입력 | 푸시 버튼 스위치 |
| PA7 | AN7 (ADC) | 입력 | 홀센서 (자석 위치 감지) |
| PD2 | LED_PWR | 출력 | 전원 LED 제어 |

## 3. 동작 설명

### 3.1 동작 흐름도

```
                    ┌──────────┐
                    │   OFF    │ ← 전원 인가 시 초기 상태
                    │ (모터정지)│
                    └────┬─────┘
                         │ 스위치 누름
                         ▼
                    ┌──────────┐
                    │    ON    │
                    │(모터 100%)│
                    └────┬─────┘
                         │ 스위치 누름
                         ▼
                    ┌──────────┐
                    │ 정지대기  │ ← 모터 계속 회전
                    │(홀센서대기)│
                    └────┬─────┘
                         │ 홀센서 감지 (ADC > 500)
                         ▼
                    ┌──────────┐
                    │   OFF    │ ← 모터 즉시 정지
                    │ (모터정지)│
                    └──────────┘
```

### 3.2 상태별 설명

| 상태 | 플래그 값 | 모터 | 설명 |
|------|----------|------|------|
| **OFF** | `0x00` | 정지 (PWM 0%) | 대기 상태. 스위치 입력 대기 |
| **ON** | `0x01` | 동작 (PWM 100%) | 모터 전속 회전. 스위치 입력 대기 |
| **정지대기** | `0x03` | 동작 (PWM 100%) | 모터 회전 유지하며 홀센서 신호 대기 |

### 3.3 스위치 동작

| 키 종류 | 시간 | 동작 |
|---------|------|------|
| **짧은 누름** | 30ms 이상 | ON/OFF 토글 |
| **긴 누름** | 5초 이상 | ON/OFF 토글 (동일) |

- **OFF 상태에서 스위치** → 모터 즉시 ON (100%)
- **ON 상태에서 스위치** → 정지대기 상태 진입 (모터 계속 회전)
- **정지대기 중 홀센서 감지** → 모터 즉시 OFF

### 3.4 홀센서 (위치 감지)

- **입력**: PA7 (ADC 채널 7, 12-bit)
- **원리**: 자석이 가까울수록 ADC 값 상승
- **감지 임계값**: `500` (12-bit 기준, 약 0.40V@3.3V)
- **용도**: 커터 칼날이 정위치(홈 포지션)에 도달했을 때 정지

## 4. 하드웨어 보호 기능

### 4.1 워치독 타이머 (WDT)

| 항목 | 값 |
|------|-----|
| 타임아웃 | **3초** |
| 클럭 | WDTRC 40kHz / 256분주 = 156Hz |
| DR | 468 (468/156 = 3초) |
| 동작 | 타임아웃 시 하드웨어 리셋 |

- 메인 루프(`Process()`) 실행 시마다 WDT 킥
- MCU 행(Hang) 발생 시 **3초 내 자동 리셋**

### 4.2 HardFault 처리

- HardFault 예외 발생 시 `NVIC_SystemReset()` 호출
- 무한루프 대신 **즉시 시스템 리셋**

## 5. PWM 사양

| 항목 | 값 |
|------|-----|
| PWM 주파수 | 20kHz |
| 타이머 | TIMER30 (MPWM 모드) |
| 클럭 | HCLK 40MHz / 10분주 = 4MHz |
| 채널 | PWM30AA (PE0) |
| 듀티 범위 | 0~100% |

## 6. 소프트웨어 타이머

| 타이머 | 주기 | 용도 |
|--------|------|------|
| TIMER10 | 1ms | 메인 루프 틱, 키 입력 처리, ADC 제어 |
| TIMER30 | 20kHz | 모터 PWM 출력 |

## 7. 파일 구조

```
source/
├── App/
│   ├── main.c              # 메인 함수, 주변장치 초기화, WDT 설정
│   ├── main.h              # 메인 헤더
│   ├── main_conf.h         # 주변장치 HAL 드라이버 include
│   ├── function.c          # Process(), Key_Input(), ADC 처리
│   ├── function.h          # UART 매크로 (미사용)
│   ├── ks_Assignment.h     # 핀 할당 정의 (DPI_KEY, ADC채널 등)
│   ├── ks_BITBAND.h        # 비트밴드 매크로
│   ├── A31G11x_PortInit.c  # GPIO 포트 초기화
│   ├── A31G11x_SystemClock.c # 시스템 클럭 설정 (HSI 40MHz)
│   └── A31G11x_it.c        # 인터럽트 핸들러
├── Device/
│   ├── A31G11x.h           # MCU 레지스터 정의
│   ├── startup_A31G11x_iar.s # IAR 스타트업 코드
│   └── system_A31G11x.c    # 시스템 초기화
└── Drivers/
    ├── Include/            # HAL 드라이버 헤더
    └── Source/             # HAL 드라이버 소스
```

## 8. 빌드 방법

1. **IAR Embedded Workbench 7.5** 에서 `A31G11x.eww` 워크스페이스 열기
2. `Debug` 구성으로 빌드 (Make)
3. 출력: `Debug/Exe/A31G11x.out`

### 커맨드라인 빌드

```
"C:\Program Files (x86)\IAR Systems\Embedded Workbench 7.5\common\bin\IarBuild.exe" A31G11x.ewp -make Debug
```
